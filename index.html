# Trellis LMS Trellis is a Learning Management System built by [Learning.net](http://learning.net). ## System Requirements - PHP 5.3.2+ - Apache 2.2+ - MySQL 4.1+ ## Installing - Install [Composer](http://getcomposer.org) and place the executable somewhere in your `$PATH` - Clone repository (we'll clone it into /var/www/trellis/ in the following examples) - Install/update the required dependencies ```bash $ cd /var/www/trellis/trellis $ composer install ```
